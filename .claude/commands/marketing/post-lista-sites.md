# Skill: Post de Lista de Sites — LinkedIn

## Quando usar

Quando o usuário pedir um post de lista de sites, links ou plataformas úteis para busca de emprego no LinkedIn. É o formato de maior alcance orgânico testado pelo JP Bradley.

## O que funciona vs. o que não funciona

**FUNCIONA:**
- Listas de sites/plataformas para encontrar vagas (o formato raiz)
- Listas de sites por nicho (remotas, internacionais, estágio, tecnologia)
- Listas de sites para ganhar em moeda estrangeira

**NÃO FUNCIONA (testado e descartado):**
- Listas de extensões do Chrome
- Listas de sites para consultar salários
- Qualquer lista que não entregue links diretos para encontrar emprego

## Estrutura obrigatória

```
NÚMERO + TÍTULO EM CAIXA ALTA (o que a lista entrega, com diferencial no parêntese ou complemento)

Gancho de 1-2 frases: problema que a pessoa tem + por que essa lista resolve

1️⃣ Nome do site — URL (diferencial em uma linha, o que esse site tem que os outros não têm)
2️⃣ Nome do site — URL (diferencial)
...

Se você gostou do conteúdo, curta esse post pra alcançar mais pessoas 👍
```

## Regras inegociáveis

- Título SEMPRE em caixa alta com número no começo
- URL sempre junto do item, nunca separada
- Descrição de cada item: 1 linha focada no DIFERENCIAL, não na descrição genérica
- CTA sempre: "Se você gostou do conteúdo, curta esse post pra alcançar mais pessoas 👍"
- Nunca usar "Compartilhe esse conteúdo" — CTA obrigatório é o de curtida com 👍
- Sem hashtags
- Sem travessão (—) em nenhum lugar do post
- Gancho sem reticências no meio

## Ângulos validados (usar como base para novos posts)

1. Vagas fora do LinkedIn e Gupy (nacional)
2. Vagas remotas globais
3. Vagas em dólar/euro (EUA, Canadá, Europa)
4. Vagas por nicho (tecnologia, marketing, estágio/trainee)
5. Sites que analisam perfil LinkedIn

## Ângulos ainda não usados (sugerir ao usuário)

- Sites de vagas para profissionais +40 anos
- Sites de vagas para quem está em transição de carreira
- Sites de vagas para trabalho por projeto (freelance qualificado)
- Sites de vagas para área de saúde fora dos portais comuns
- Sites de vagas para trabalho híbrido no Brasil
